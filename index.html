Hybrid Computational Models of Spinal Cord Stimulation

Technologies: MATLAB, Python, COMSOL Multiphysics, NEURON

Developed a hybrid computational model of the thoracic spinal cord to study neural recruitment under transcutaneous and epidural stimulation.

The model integrates finite element simulations (COMSOL) with neural dynamics (NEURON) to analyze how different stimulation patterns affect selectivity and activation of spinal circuits.

  • Built a multi-scale simulation pipeline combining physics and neural modeling
  • Analyzed recruitment and selectivity across stimulation configurations
  • Collaborated with clinicians to ensure translational relevance

Task Prediction Using fMRI Data

Technologies: Python, PyTorch, RNNs, CNNs

Developed deep learning models to predict task timecourses directly from fMRI brain activity using data from the Human Connectome Project.

  • Implemented recurrent neural networks to capture temporal dynamics
  • Evaluated model performance on real neuroimaging datasets

MERLOT — Data Story on Online Content & Network Structure

Technologies: Python, Network Analysis, NLP, Data Visualization

This project explores a large-scale dataset of online content and interactions, focusing on how information spreads and how relationships form in digital ecosystems.

The goal was to extract meaningful structure from complex, noisy data and turn it into an interpretable data story.

  • Built and analyzed large-scale interaction networks
  • Applied text processing and feature extraction techniques
  • Studied community structure and connectivity patterns
  • Designed visualizations to communicate insights clearly

The project emphasizes the link between **structure (networks)** and **content (textual signals)** in real-world datasets.
